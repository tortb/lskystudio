### C. 命令参考

```bash
# 开发
pnpm dev                          # 启动前端开发服务器
pnpm tauri dev                    # 启动 Tauri 开发模式

# 构建
pnpm build                        # 构建前端
pnpm tauri build                  # 打包 Tauri 应用

# 测试
pnpm test                         # 运行测试
pnpm lint                         # 代码检查
pnpm format                       # 代码格式化
```
